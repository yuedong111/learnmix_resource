# 九、货币政策类术语

## 48. 货币政策

货币政策是央行通过控制货币和信贷条件来影响经济。

货币政策主体是：

> 中央银行。

例如中国人民银行、美联储、欧洲央行、日本央行。

---

## 49. 中央银行

中央银行是管理货币和金融体系的机构。

主要职责包括：

| 职责     | 含义       |
| ------ | -------- |
| 发行货币   | 管理基础货币   |
| 调控利率   | 影响借贷成本   |
| 管理银行体系 | 维护金融稳定   |
| 控制通胀   | 保持物价稳定   |
| 外汇储备管理 | 维护国际支付能力 |

---

## 50. 利率

利率是借钱的价格。

利率高：

> 借钱贵，消费和投资下降。

利率低：

> 借钱便宜，消费和投资上升。

所以央行常通过利率调控经济冷热。

---

## 51. 降息

降息就是降低利率。

作用：

| 对象 | 影响           |
| -- | ------------ |
| 企业 | 融资成本降低，更愿意投资 |
| 居民 | 房贷、消费贷成本降低   |
| 股市 | 资金更可能流入资产市场  |
| 汇率 | 本币可能贬值压力增加   |

降息通常用于刺激经济。

---

## 52. 加息

加息就是提高利率。

作用：

| 对象 | 影响        |
| -- | --------- |
| 企业 | 融资成本上升    |
| 居民 | 贷款压力增加    |
| 消费 | 可能下降      |
| 通胀 | 可能被压制     |
| 汇率 | 本币可能更有吸引力 |

加息通常用于抑制通胀或资产泡沫。

---

## 53. 存款准备金率

存款准备金率是商业银行必须留在央行的钱的比例。

比如银行吸收 100 元存款，如果准备金率是 10%，就必须留 10 元，最多只能贷出 90 元。

准备金率越高，银行能放贷的钱越少。
准备金率越低，银行能放贷的钱越多。

---

## 54. 降准

降准就是降低存款准备金率。

效果是：

> 释放银行资金，让银行有更多钱可以贷款。

通常用于宽松货币条件，支持经济。

---

## 55. 公开市场操作

公开市场操作是央行买卖政府债券来调节市场流动性。

央行买入债券：

> 向市场投放货币。

央行卖出债券：

> 从市场回收货币。

---

## 56. 量化宽松，QE

量化宽松是央行在利率已经很低时，进一步大量购买债券等资产，向市场注入流动性。

简单理解：

> 央行直接往金融系统里加大量的钱。

QE 常见于金融危机或严重衰退时期。

---

## 57. 量化紧缩，QT

量化紧缩是 QE 的反方向。

央行减少资产负债表规模，回收流动性。

作用：

> 收紧金融环境，压制通胀或资产泡沫。

---

## 58. 流动性

流动性可以理解为：

> 市场上钱是否充足，资产是否容易变现。

流动性充足时：

> 借钱容易，资产价格容易上涨。

流动性紧张时：

> 借钱困难，企业和金融市场压力加大。

---

# 十、货币与银行类术语

## 59. 货币供应量

货币供应量指经济体中流通的钱的总量。

常见口径有 M0、M1、M2。

---

## 60. M0

M0 是流通中的现金。

比如纸币、硬币。

---

## 61. M1

M1 通常包括：

> 现金 + 活期存款

它代表经济中非常活跃、随时能用的钱。

M1 增长快，通常说明交易活动比较活跃。

---

## 62. M2

M2 通常包括：

> M1 + 定期存款 + 储蓄存款等

M2 是更广义的钱。

M2 增长快，不一定代表经济好，还要看钱有没有进入实体经济。

如果钱只是在金融系统或房地产里转，实体经济可能仍然疲弱。

---

## 63. 信贷

信贷就是银行或金融机构把钱借出去。

经济很多时候不是单纯靠“已有的钱”，而是靠信用创造出来的购买力。

例如：

企业贷款扩产，居民贷款买房，都会创造需求。

---

## 64. 信用扩张

信用扩张指银行贷款增加，社会融资变多。

它会推动：

| 领域   | 影响        |
| ---- | --------- |
| 消费   | 居民贷款消费    |
| 投资   | 企业贷款扩张    |
| 房地产  | 房贷增加推高需求  |
| 资产价格 | 股市、楼市可能上涨 |

---

## 65. 信用收缩

信用收缩指贷款减少、融资变难。

表现为：

> 企业借不到钱，居民不敢贷款，银行惜贷。

信用收缩严重时，经济会快速降温。

---

## 66. 杠杆率

杠杆率表示债务相对于收入或资产的比例。

例如：

一个人年收入 20 万，却背了 200 万贷款，杠杆就很高。

整个经济杠杆率高，说明债务压力大，对利率和收入变化更敏感。

---

## 67. 去杠杆

去杠杆指降低债务水平。

方式包括：

| 方式    | 说明       |
| ----- | -------- |
| 还债    | 主动降低负债   |
| 债务重组  | 延期、降息、减免 |
| 减少新贷款 | 限制信用扩张   |
| 破产清算  | 无力偿还时出清  |

去杠杆短期会压低经济增长，但长期可能降低风险。

---

# 十一、金融市场类术语

## 68. 债券

债券本质是借条。

政府或企业发行债券，就是向投资者借钱，并承诺支付利息。

---

## 69. 国债

国债是政府发行的债券。

通常被认为风险较低。

国债收益率是宏观经济中非常重要的指标。

---

## 70. 债券收益率

债券收益率可以理解为持有债券能获得的回报率。

债券价格和收益率通常反向变动：

| 债券价格 | 收益率 |
| ---- | --- |
| 上涨   | 下降  |
| 下跌   | 上升  |

---

## 71. 收益率曲线

收益率曲线显示不同期限债券的收益率。

正常情况下：

> 长期利率高于短期利率。

因为借长期钱风险更大。

---

## 72. 收益率曲线倒挂

收益率曲线倒挂指短期利率高于长期利率。

这通常被视为经济衰退风险信号。

因为市场预期未来经济会变差，长期利率会下降。

---

## 73. 资产泡沫

资产泡沫指资产价格明显高于其基本价值。

比如：

房价、股票、加密资产价格被过度炒作。

泡沫形成通常依赖：

> 低利率、宽信用、乐观预期、杠杆资金。

---

## 74. 金融危机

金融危机是金融系统出现严重失灵。

表现包括：

| 表现   | 说明      |
| ---- | ------- |
| 银行挤兑 | 大量储户取钱  |
| 信贷冻结 | 银行不敢贷款  |
| 资产暴跌 | 房价、股价下跌 |
| 企业破产 | 融资断裂    |
| 失业上升 | 实体经济受冲击 |

---

# 十二、开放经济类术语

## 75. 汇率

汇率是一种货币兑换另一种货币的价格。

比如：

> 1 美元 = 7 元人民币

这就是美元兑人民币汇率。

---

## 76. 本币升值

本币升值指本国货币变贵。

影响：

| 方面   | 结果    |
| ---- | ----- |
| 进口   | 更便宜   |
| 出口   | 竞争力下降 |
| 出国消费 | 更便宜   |
| 外资流入 | 可能增加  |

---

## 77. 本币贬值

本币贬值指本国货币变便宜。

影响：

| 方面   | 结果        |
| ---- | --------- |
| 出口   | 更有竞争力     |
| 进口   | 更贵        |
| 留学旅游 | 成本上升      |
| 通胀   | 输入性通胀压力增加 |

---

## 78. 国际收支

国际收支记录一个国家与世界其他国家之间的经济交易。

包括：

| 项目      | 内容            |
| ------- | ------------- |
| 经常账户    | 商品、服务、收入、转移支付 |
| 资本和金融账户 | 投资、资本流动       |
| 外汇储备    | 央行持有的外币资产     |

---

## 79. 经常账户

经常账户主要包括贸易收支。

如果一个国家出口大于进口，通常经常账户顺差。

如果进口大于出口，通常经常账户逆差。

---

## 80. 贸易顺差

贸易顺差指：

> 出口 > 进口

说明一个国家向外卖出的商品和服务多于买入的商品和服务。

---

## 81. 贸易逆差

贸易逆差指：

> 进口 > 出口

说明一个国家买入外国商品服务多于卖出商品服务。

---

## 82. 外汇储备

外汇储备是央行持有的外币资产。

主要用于：

| 用途   | 说明        |
| ---- | --------- |
| 稳定汇率 | 必要时干预市场   |
| 国际支付 | 支付进口和外债   |
| 增强信心 | 证明国家有偿付能力 |
| 应对危机 | 防止资本外逃冲击  |

---

## 83. 资本流动

资本流动指资金跨国流入或流出。

资本流入：

> 外资进入本国股市、债市、房地产、企业投资。

资本流出：

> 本国资金流向海外。

资本快速流出可能导致汇率贬值和金融市场动荡。

---

# 十三、收入分配与福利类术语

## 84. 收入分配

收入分配研究社会总收入如何在不同群体之间分配。

比如：

资本所有者、劳动者、高收入群体、低收入群体之间如何分配。

---

## 85. 基尼系数

基尼系数衡量收入不平等程度。

范围通常在 0 到 1 之间。

| 基尼系数  | 含义   |
| ----- | ---- |
| 越接近 0 | 越平等  |
| 越接近 1 | 越不平等 |

---

## 86. 贫困率

贫困率表示低于某个贫困标准的人口比例。

它衡量社会中有多少人无法维持基本生活水平。

---

## 87. 中等收入陷阱

中等收入陷阱指一个国家从低收入进入中等收入后，无法继续升级为高收入国家。

原因可能包括：

| 原因       | 说明          |
| -------- | ----------- |
| 低端制造优势消失 | 工资上涨，成本优势下降 |
| 技术创新不足   | 无法进入高附加值产业  |
| 教育质量不足   | 人力资本跟不上     |
| 制度改革滞后   | 资源配置效率低     |
| 贫富差距扩大   | 内需不足        |

---

# 十四、宏观政策目标类术语

## 88. 充分就业

充分就业不是所有人都有工作，而是失业率接近自然失业率。

也就是：

> 想工作的人基本能找到合适工作。

---

## 89. 物价稳定

物价稳定不是物价不变，而是通胀保持在温和可控水平。

很多国家央行会把通胀目标设在大约 2% 左右。

---

## 90. 经济增长

政府希望经济长期稳定增长，提高居民收入和生活水平。

---

## 91. 国际收支平衡

国际收支平衡指一个国家对外经济关系比较稳定，不出现严重外债、汇率或资本流动危机。

---

## 92. 金融稳定

金融稳定指银行、资本市场、支付系统正常运行，不发生系统性危机。

金融稳定非常重要，因为金融系统一旦出问题，会迅速传导到实体经济。

---

# 十五、常见宏观经济模型术语

## 93. 凯恩斯主义

凯恩斯主义认为：

> 经济衰退时，问题往往是总需求不足，政府应该积极干预。

它强调财政政策和政府支出。

典型思路：

> 居民不消费，企业不投资，政府就要站出来扩大需求。

---

## 94. 古典经济学

古典经济学更强调市场自我调节。

认为价格、工资、利率会调整，使经济回到均衡。

它更相信市场机制。

---

## 95. 货币主义

货币主义强调货币供应量对经济和通胀的重要影响。

代表人物是弗里德曼。

其经典观点：

> 通货膨胀归根到底是一种货币现象。

---

## 96. 新古典经济学

新古典经济学强调理性预期、市场出清和个体最优化。

它认为政策效果会受到人们预期的影响。

---

## 97. 新凯恩斯主义

新凯恩斯主义承认市场有缺陷，比如价格粘性、工资粘性、信息不完全。

因此它认为政府和央行政策仍然有作用。

现代主流宏观经济学很多内容都属于新凯恩斯框架。

---

## 98. IS-LM 模型

IS-LM 模型解释商品市场和货币市场如何共同决定收入和利率。

简单理解：

| 曲线    | 含义     |
| ----- | ------ |
| IS 曲线 | 商品市场均衡 |
| LM 曲线 | 货币市场均衡 |

它常用于分析财政政策和货币政策的短期影响。

---

## 99. AD-AS 模型

AD-AS 模型是总需求和总供给模型。

它解释：

> 产出和物价水平如何由总需求、总供给共同决定。

可以用来分析通胀、衰退、供给冲击等问题。

---

## 100. 菲利普斯曲线

菲利普斯曲线描述通胀和失业之间的关系。

传统观点认为：

> 失业率低时，通胀压力大；失业率高时，通胀压力小。

但长期来看，这种关系并不稳定。

---
