
# Chapter 3: 证据归档模板

上一章我们学会了检查租房合同的“核心条款”，但就算合同写得再清楚，如果没有证据，纠纷发生时还是可能“空口无凭”。比如租房时，房东说“房屋有划痕，不退押金”，但如果你没拍入住前的照片，就很难证明划痕是之前就有的。这一章我们要解决的问题是：**如何系统保存有效证据，让纠纷发生时“有据可查”**。


## 1. 为什么需要“证据归档模板”？
想象一下：小张借给朋友5000元，朋友说“下周还”，但后来没还。小张想维权，但只有微信聊天记录，没有转账备注，朋友不承认是借款。这时候，如果小张提前保存了“转账备注写‘借款5000元’”和“聊天记录确认还款时间”，就能轻松证明。证据归档模板就像一本“证据手册”，把聊天记录、转账备注、照片、合同等整理好，纠纷发生时不是“空口无凭”，而是有据可查。


## 2. 什么是证据归档模板？
证据归档模板是**系统保存有效证据的方法**，包括聊天记录、转账备注、照片、合同等。它的核心是：**把可能产生纠纷的证据，提前分类整理，方便后续使用**。比如租房时，你要保存“入住前房屋照片”“押金转账备注”“合同”“聊天记录”等；借钱时，要保存“转账备注”“聊天确认借款”等。


## 3. 需要保存哪些证据？怎么保存？
证据归档不是“随便存”，而是要保存**关键场景下的核心证据**。以下是常见场景的“证据清单”：

| 场景         | 需要保存的证据                                                                 | 保存方法                                                                 |
| ------------ | ------------------------------------------------------------------------ | ------------------------------------------------------------------------ |
| **借钱/还款**   | 转账记录（备注写“借款5000元”）、聊天记录（确认借款金额、还款时间）、借条（可选）             | 转账备注要明确，聊天记录不要删除，借条拍照保存。                                   |
| **租房**       | 入住前房屋照片/视频、押金转账备注（写“XX房屋押金”）、租房合同、家具家电清单、聊天记录（沟通维修） | 照片/视频拍清楚房屋状况，转账备注写清楚用途，合同和清单拍照保存。                         |
| **购物/消费**   | 发票/收据、聊天记录（确认商品/服务）、转账记录（备注写“购买XX商品”）                     | 发票拍照，聊天记录保存，转账备注明确。                                           |
| **工作/合同**   | 劳动合同、工资条、社保缴纳记录、聊天记录（确认加班/薪资）                             | 合同和工资条拍照，社保记录截图，聊天记录保存。                                     |


## 4. 举个例子：租房押金纠纷怎么用证据归档？
假设你退租时，房东说“房屋有划痕，不退押金”，但你有证据归档模板，里面保存了：

1. **入住前照片**：拍清楚墙面、家具，证明划痕是之前就有的；
2. **押金转账备注**：写“XX房屋押金”，证明押金金额和用途；
3. **租房合同**：写清楚“押金2000元，退租后7天内退还（房屋完好）”；
4. **聊天记录**：和房东沟通“入住时墙面有划痕，已拍照”的记录。

这时候，你可以拿出这些证据，告诉房东：“划痕是入住前就有的，合同里写清楚房屋完好，所以押金应该退还。”房东看到证据，很难再找理由拒绝。


## 5. 内部实现：证据归档的流程是怎样的？
当你需要保存证据时，其实是一个“收集-分类-保存”的过程。用简单的流程图展示：

```mermaid
sequenceDiagram
    participant 你 as 当事人
    participant 手机 as 手机/电脑
    你->>手机： 收集证据（拍照、截图、保存聊天）
    你->>你： 分类整理（按场景分：租房、借钱、工作）
    你->>手机： 保存归档（存到固定文件夹，比如“证据-租房”）
    你->>你： 需要时使用（纠纷发生时，拿出对应证据）
```

简单来说，**证据归档就是“提前整理，避免临时慌乱”**。比如租房时，入住前拍照片，转账时写备注，这些小事做对了，纠纷时就有底气。


## 6. 常见坑：这些情况要注意！
- 转账备注不写：借钱时只转钱，没写“借款”，对方不承认；
- 照片没拍清楚：租房时没拍墙面，房东说“划痕是你弄的”；
- 聊天记录删除：和房东沟通维修时，删除了聊天记录，没证据；
- 合同没保存：租房合同丢了，无法证明押金退还条件。

- 保存方法要简单（拍照、截图、写备注）；
