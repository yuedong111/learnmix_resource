
# Chapter 1: 劳动合同必备条款

## 1. 为什么要注意劳动合同的“骨架”？
想象一下：小张入职一家公司，HR说“试用期月薪8000，转正后10000”，但他没仔细看合同，结果合同里只写了“薪资按公司规定执行”。3个月后，他发现实际到手只有6000，社保也没按真实工资缴。这时候，他才发现合同里没写清楚薪资结构、社保基数，维权时连证据都没有。

劳动合同就像房子的“骨架”——如果骨架不完整，你的权益（比如薪资、社保、试用期）就容易被“偷工减料”。今天我们就来拆解这个“骨架”，让你知道哪些条款必须写清楚。


## 2. 什么是劳动合同必备条款？
劳动合同必备条款是《劳动合同法》规定必须包含的核心内容，就像一份“合同骨架”。缺少这些条款，你的劳动权益可能无法得到保障（比如试用期、违约责任等）。简单来说，就是**合同里必须有的“硬性规定”**。


## 3. 必备条款有哪些？怎么检查？
拿到合同后，先别急着签字，逐项核对下面的“骨架”是否齐全：

| 必备条款       | 检查要点                                                                 |
| ------------ | ------------------------------------------------------------------------ |
| **公司名称**   | 是否写全称（比如“XX科技有限公司”而不是“某公司”）？避免模糊主体。                     |
| **你的岗位**   | 是否明确（比如“销售经理”而不是“其他”）？避免被随意调岗。                           |
| **工作地点**   | 是否写具体城市/区域（比如“上海市浦东新区”）？避免被派到偏远地方。                     |
| **工作时间**   | 是否写清楚每天/每周工作时长（比如“每天8小时，每周5天”）？避免加班没补偿。               |
| **休息休假**   | 是否写明年假、病假、产假等假期天数？避免公司不让你休。                             |
| **劳动报酬**   | 是否写清楚税前/税后薪资？结构是什么（比如“基本工资5000+绩效3000”）？避免口头承诺不兑现。   |
| **社会保险**   | 是否写明缴纳“五险”（养老、医疗、失业、工伤、生育）？是否按真实工资缴纳？避免按最低基数缴。   |
| **合同期限**   | 是否写清楚是固定期限（比如1年）还是无固定期限？避免试用期过长。                       |
| **试用期**     | 是否写清楚期限（比如“3个月”）？是否符合法律规定（比如1年合同试用期不超过1个月）？         |
| **违约责任**   | 是否写清楚违约赔偿（比如“提前离职需赔偿培训费”）？避免被不合理索赔。                   |


## 4. 举个例子：怎么用这些条款解决问题？
假设你拿到一份合同，发现“劳动报酬”只写了“月薪8000”，没写税前/税后，也没写结构。这时候你应该：

1. **提出疑问**：问HR“薪资是税前还是税后？有没有绩效、补贴？”
2. **要求修改**：让HR把薪资结构写进合同（比如“税前月薪8000，其中基本工资5000，绩效3000”）。
3. **拒绝签字**：如果HR不同意修改，先别签，避免后续纠纷。


## 5. 内部实现：合同签订的流程是怎样的？
当你拿到合同后，其实是一个“核对-沟通-修改-签字”的过程。用简单的流程图展示：

```mermaid
sequenceDiagram
    participant 你 as 员工
    participant 公司 as HR
    你->>公司： 拿到劳动合同
    你->>你： 逐项核对必备条款
    你->>公司： 提出疑问（比如“薪资结构没写清楚”）
    公司->>你： 修改合同
    你->>你： 再次核对修改后的条款
    你->>公司： 签字确认
```

简单来说，**合同不是“HR给什么就签什么”，而是你主动核对、沟通后的结果**。


## 6. 常见坑：这些情况要注意！
- 试用期不签合同：法律规定试用期必须签合同，否则视为无试用期。
- 薪资口头承诺：口头说“转正后加薪”，但合同里没写，后续维权没证据。
- 社保按最低基数缴：公司为了省钱，按最低工资缴社保，影响你的养老金、医保报销。
- 试用期过长：1年合同试用期不能超过1个月，3年合同不能超过2个月。
