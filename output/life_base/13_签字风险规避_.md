
# Chapter 13: 签字风险规避

上一章我们学会了算清楚每月工资到手多少（[工资到手计算表](12_工资到手计算表.md)），但就算工资算对了，如果签字时没注意，可能还是会掉进“签字陷阱”——比如签了空白离职协议，结果被要求放弃经济补偿；或者替朋友签了担保书，朋友逾期不还，你得替他还钱。这一章的“签字风险规避”，就像一份“签字风险探测器”，帮你识别哪些文件不能随便签，避免承担不必要的责任。


## 1. 为什么签字要“小心”？
想象一下：小张离职时，HR递来一份“离职协议”，说“签了就能结清工资”。小张没仔细看，直接签了。结果协议里写了“自愿放弃经济补偿”，后来小张才发现自己少拿了2个月工资，但合同已经签了，维权时很难推翻。  
签字不是“对方给什么就签什么”，而是你主动检查、沟通后的结果。如果签了对自己不利的文件，就像“自己给自己挖坑”，后续很难补救。


## 2. 什么是“签字风险”？
签字风险就是**签了对自己不利的文件**，比如：
- 文件有空白条款（比如没填内容的合同，对方可以事后补写）；
- 文件有“放弃权利”的条款（比如“自愿放弃社保”“自愿放弃补偿”）；
- 你可能不懂条款含义（比如担保书里的“连带责任”）。

简单来说，**签字风险 = 签了“可能让你吃亏”的文件**。

| **空白合同**       | 合同内容没填，对方可以事后补写（比如空白离职协议，对方写“自愿放弃补偿”）。                     | 小张签了空白离职协议，后来被写上“不拿经济补偿”。                                   |

## 4. 怎么用“签字风险规避”避免风险？
拿到文件时，按以下步骤检查，像“签字前的安全检查”：

### 步骤1：看有没有“空白条款”
如果文件有空白处（比如合同没填薪资、离职补偿），**先让对方填清楚**，再签字。  
比如：离职协议里有“经济补偿：________元”，你要让HR填具体金额，或者写“按《劳动合同法》规定计算”。

### 步骤2：看有没有“放弃权利”的条款
如果文件里有“自愿放弃社保”“自愿放弃补偿”等条款，**问清楚为什么**，或者拒绝签字。  
比如：HR让你签“放弃社保协议”，你要问：“社保是法定义务，为什么放弃？如果放弃，我的医保、养老金怎么办？”

### 步骤3：不懂的条款不要签
如果条款你看不懂（比如担保书里的“连带责任”），**要求对方解释清楚**，或者找专业人士帮忙。  
比如：朋友让你签担保书，你不懂“连带责任”，可以问律师：“如果我签了，万一朋友不还钱，我要承担什么责任？”


## 5. 举个例子：小张怎么用“签字风险规避”避免坑？
小张离职时，HR递来一份离职协议，里面有“自愿放弃经济补偿”的条款。小张按以下步骤检查：

1. **看空白条款**：协议里“经济补偿”是空白的，小张要求HR填“按《劳动合同法》规定计算（2个月工资）”。  
2. **看放弃权利条款**：协议里有“自愿放弃经济补偿”，小张问HR：“为什么放弃？法律规定离职要给补偿，我该拿多少？”  
3. **不懂不签**：小张不懂“经济补偿”怎么算，让HR解释：“根据我的工作年限，应该给2个月工资，对吗？”  

最终，HR修改了协议，小张签了字，拿到了应得的补偿。

def check_signing_risk(file_content):
    """
    检查文件是否有签字风险（空白条款或放弃权利条款）
    :param file_content: 文件内容（字符串）
    :return: 风险列表（比如["有空白条款", "有放弃权利条款"]）
    """
    risks = []
    # 检查空白条款（比如“________”或“______”）
    if "________" in file_content or "______" in file_content:
        risks.append("有空白条款，需让对方填清楚")

        risks.append("有放弃权利条款，需问清楚原因")
    return risks
