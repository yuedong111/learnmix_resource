
# Chapter 12: 工资到手计算表

上一章我们学会了退租时如何按流程操作，顺利拿回押金（[退租流程规范](11_退租流程规范.md)）。这一章我们要解决工作中最常见的问题之一——**工资到手计算表**，帮你算清楚每月实际能拿到多少钱，避免被“模糊薪资”坑（比如绩效占比过高导致到手不稳定）。


## 1. 为什么需要“工资到手计算表”？
想象一下：小张入职一家公司，HR说“月薪10000元”，但他没算清楚扣除项，结果第一个月到手只有7500元。他问HR：“为什么少这么多？”HR说：“扣了社保、公积金、个税，还有绩效没发。”小张这才意识到，原来“月薪10000”不是“到手10000”。  
“工资到手计算表”就像一份“薪资翻译器”，把税前工资、扣除项、到手工资拆解清楚，让你知道每月实际能拿到多少钱，避免被“模糊薪资”坑。


## 2. 核心概念：税前工资 vs 到手工资
要算清楚到手工资，先得明白两个关键概念：


### （1）税前工资：你的“总蛋糕”
税前工资是公司给你发的“总金额”，包括基本工资、绩效工资、补贴、加班费等。比如小张的税前工资是10000元，其中基本工资5000元，绩效3000元，补贴2000元。


### （2）扣除项：从“总蛋糕”里切掉的部分
扣除项是公司从税前工资里扣的钱，主要包括：  
- **社保个人部分**：养老、医疗、失业、工伤、生育保险的个人缴费（比如小张的社保个人部分是1000元）；  
- **公积金个人部分**：住房公积金的个人缴费（比如小张的公积金个人部分是1200元）；  
- **个人所得税（个税）**：根据税前工资扣除社保、公积金后的金额计算的税（比如小张的个税是300元）。


### （3）到手工资：剩下的“蛋糕”
到手工资就是税前工资减去扣除项后的金额，是你实际能拿到手的钱。比如小张的到手工资是：  
10000（税前） - 1000（社保） - 1200（公积金） - 300（个税） = 7500元。


## 3. 怎么用“工资到手计算表”算清楚？
算到手工资很简单，分3步：**确定税前工资→计算扣除项→算到手工资**。


### 步骤1：确定税前工资
先看你的工资条或劳动合同，找到“税前工资”或“应发工资”这一项。比如：  
- 基本工资：5000元  
- 绩效工资：3000元  
- 补贴：2000元  
- 税前工资 = 5000 + 3000 + 2000 = 10000元  


### 步骤2：计算扣除项
扣除项包括社保、公积金、个税，分别计算：  
- **社保个人部分**：看工资条上的“社保个人缴费”，比如1000元；  
- **公积金个人部分**：看工资条上的“公积金个人缴费”，比如1200元；  
- **个税**：用“税前工资 - 社保 - 公积金”后的金额计算（比如10000 - 1000 - 1200 = 7800元），然后查个税表（比如7800元的个税是300元）。  


### 步骤3：算到手工资
把税前工资减去扣除项，就是到手工资：  
到手工资 = 税前工资 - 社保个人部分 - 公积金个人部分 - 个税  
比如小张的到手工资是：10000 - 1000 - 1200 - 300 = 7500元。

## 5. 常见坑：这些情况要注意！
- **绩效占比过高**：比如绩效工资占税前工资的50%，如果绩效没达标，到手工资会大幅下降（比如小张的绩效没发，到手只有5000元）；  
- **公司按最低基数缴社保**：比如小张的税前工资是10000元，但公司按3000元（当地最低基数）缴社保，导致个税算错（比如个税应该是100元，但公司按3000元算，个税变成50元，看似少扣，但社保权益缩水）；  
- **个税专项附加扣除没填**：比如小张有房贷，可以填“住房贷款利息”专项扣除，每月少交100元个税，但没填的话，多交了100元；  
- **加班费没算**：比如小张加班了10小时，每小时50元，加班费500元，但公司没算，导致税前工资少了500元。

## 7. 代码示例：用Python算到手工资
如果你喜欢用代码，可以用简单的Python函数计算到手工资，注释很清楚：

```python
def calculate_net_salary(gross_salary, social_security, housing_fund, tax):
    """
    计算到手工资
    :param gross_salary: 税前工资（元）
    :param social_security: 社保个人部分（元）
    :param housing_fund: 公积金个人部分（元）
    :param tax: 个税（元）
    :return: 到手工资（元）
    """
    net_salary = gross_salary - social_security - housing_fund - tax
    return net_salary

# 例子：小张的工资
gross_salary = 10000  # 税前工资
social_security = 1000  # 社保个人部分
housing_fund = 1200  # 公积金个人部分
tax = 300  # 个税
net_salary = calculate_net_salary(gross_salary, social_security, housing_fund, tax)
print(f"到手工资是：{net_salary}元")  # 输出：到手工资是：7500元
```
