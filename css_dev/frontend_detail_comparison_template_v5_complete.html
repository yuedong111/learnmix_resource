Developer Notes

最后这一批模块怎么抓重点

  1. Pagination 看按钮密度和 active 态，不要又厚又重。
  2. Dropdown 看 trigger 和 menu 是不是一个系统，hover/selected 是否清楚。
  3. Drawer 看 handle、上边缘圆角和上浮阴影是否自然。
  4. Stepper 看当前位置和信息说明能否帮助用户理解流程，而不是只显示数字。
  5. Calendar 看默认态、范围态、选中态能否快速区分。
  6. Upload 看拖拽区域是否真的像“可投放目标”，不是一块空白虚线框。
  7. Detail Page 看主指标、状态、补充字段有没有分层。
  8. Profile Page 看头像、身份标签、角色信息和文案是否组织成稳定结构。
