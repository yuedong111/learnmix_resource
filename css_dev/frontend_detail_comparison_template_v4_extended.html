Developer Notes

这一版新增模块怎么训练

  1. Tabs 看 active 态是不是“轻而明确”，不要像一排厚按钮。
  2. Sidebar 看导航是否安静、层级是否清楚，active 是否只是轻提示而不是重色块。
  3. Empty State 看是否有引导文案和动作，不要只写一句 “No data”。
  4. Toast / Alert 看反馈是否清楚，严重程度是否能从视觉上快速判断。
  5. Search / Filter 看高频操作区是否整洁，输入框和筛选器是否形成统一系统。
  6. Dashboard Widget 看数值、标签、辅助趋势信息有没有形成明确的信息层级。
  7. 登录页和设置页要重点看：圆角、边界、输入框、说明文案、开关控件是否都在同一视觉语言里。
