为什么要写这个教程

Claude Code本身是程序员写代码用的工具，但其实也是一个本地的通用智能体，非常适合执行本地任务，并且完成任务后文件自动放在本地文件夹。本教程面向的是非程序员，教你如何使用Claude Code做除了写代码外的其他任务。
看完本教程后能学到什么？

让Claude Code自动上网搜索资料并整理成报告；
使用Claude Code整理发票到Excel表格；
让Claude Code整理你的文件夹、分析你本地文件的内容；
使用Claude Code调用API批量免费生成图片；
更多潜力可以自己多尝试。本文不教如何使用Claude Code写复杂的代码项目，这方面可以看更专业的其他教程。
前期准备

确保之前安装的Claude Code可以正常使用，具体安装教程可以查看我的上一篇文章：https://mp.weixin.qq.com/s/9Rr4ORLGtOQjtpOBFC2CHg
如果使用时提示401错误，关注“AI加速派”，发送 cc，重新获取最新的settings.json文件，替换当前./claude文件夹里的同名文件。401错误的提示如下图。

开始使用

1、安装MCP
安装的作用是让Claude Code能联网搜索。
1.1 进入要操作的文件夹，右键启动powershell
例如我打开的是桌面上的一个文件夹，在空白处右键，点击在终端中打开。这样可以限定Claude Code的工作目录，只能在这个文件夹里读文件和写文件。
图像
1.2 输入claude --dangerously-skip-permissions，使用最高权限启动。
黑色命令行界面，输入指令：claude --dangerously-skip-permissions
输入后回车，然后按方向键的下键，选第2个，然后回车。
这个指令会给Claude Code最大权限，因为安装MCP需要各种反复确认，使用这个指令Claude Code可以直接处理，省去各种确认。
图像
1.3 开始安装MCP
复制下面的命令，粘贴到命令行，然后回车。这个命令可以安装搜索MCP，里面的Key是我的，新注册的账号大概能搜索2000次，你可以在exa.ai/申请你自己的KEY，然后跟Claude Code说修改exa MCP的Key为你的Key，他会自动修改。
claude mcp add exa -e EXA_API_KEY=db135c2b-246e-4b01-ac49-1d16b5b24b30 -- npx -y exa-mcp-server
